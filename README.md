# strapi-automation-pipeline
Automated Strapi deployment using GitHub Actions and Terraform. CI builds and pushes Docker image on code push. CD workflow provisions EC2 and deploys container manually. Uses GitHub Secrets for secure AWS access. Strapi verified via EC2 public IP and SSH. Portfolio-ready DevOps pipeline.
